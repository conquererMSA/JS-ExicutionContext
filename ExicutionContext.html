How JS runs your code

Interpreter: Amader lekha human readable code instruction er protiti equivalent machine instruction browser engine e lekha ache. Interpreter line by line exicute kore. Mane protom line interprete kore, tarpor abar code fire eshe ditio line browser engine e niye zay ebong engine machine code e convert kore. Interpretation process onek slow chilo, kintu easy to debug.

Compilation: Compilation e pur file er code eksathe browser engine machine code e convert kore. Compilation e line by line process chilo na, ejonno compilation onek fast chilo. Kintu code er moddhye khane error hole compilation bondo takto na, compilation complete hoye zeto, ejonno kun line e error hoyeche ta khuje ber kora one kothin chilo. Thik e karonei Compilation process hard to debug chilo.

JIT-Compiler: JIT: Interprete kore+ Compiler: code ke machine code e convert kore. Zokhon kuno fuction call hoy tokhon just in time er moddhye oi function er code machine code e convert hoy ebong run hoy.
